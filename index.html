Quick help

  • Keyboard: use digits, +, -, *, /, Enter, Backspace, Escape.
  • Percent: converts the current value into a percentage. For + and -, it uses the stored value as the base.
  • Restore: click a history card to move its result back to the display.

The calculator keeps the latest results in local storage so they remain available after refresh.
